Primeiras linguagens de programação

Aqui voce é apresentado as primeiras linguagens e a mais utilizadas por programadores que estao iniciando neste novo mundo. O intuito principal é apresentar e explicar de maneira breve o que cada linguagem representa e como elas se conectam entre si. Antes de aprender qualquer linguagem, você precisa aprender a pensar.

Logica de programação

Lógica de programação é como ensinar o computador a entender decisões "se isso acontecer, faça aquilo". Não é código ainda, é raciocinio.

HTML

HTML é a base e o esquelo de qualquer site. Tudo que forma um site: Títulos, Texto, Imagens, Links. sem HTML não existe site, ele é o ponto inicial de todo site.

Exemplo:


            <h1>Olá, mundo!</h1>
            <p>muito obrigado por visitar meu site.</p>
            <a ref="#">clique aqui</a>

CSS

O CSS por outro lado é totalmente responsavel pela aparencia. Cores, Fontes, Tamanhos, Animações e organização visual. E ele que torna um site simples em algo visualmente bonito.

Exemplo:


            body {
                background: black;
                color: cyan;
            }
            h1 {
                color: violet
            }

Python

Python é conhecida por ser uma linguagem de fácil entendimento e leitura, mas é usada em áreas avançadas como inteligência artificial, automação e ciência de dados.

Exemplo:


            print("salve salve, mundo");

JavaScript

JavaScript é oque faz a pagina reagir. É o que traz "vida" ao site Botões funcionais, Formulários, Animações e todas as interações do site sao responsabilidade do JavaScript.

Exemplo:


            alert("Bem vindo ao site");

C++

C++ é usada na estruturação e codificação de jogos, sistemas, motores graficos e softwares pesados. É uma linguagem mais avançada que exige maior conhecimento sobre logica de programação e controle maior sobre os comandos do computador.

Exemplo:


          #include 
            using name space std;

            int main () {
                count << "Ola, mundo!";
                return 0;
            }

C#

C# é muito usada na criação de jogos (unity), softwares Windows e aplicações corporativas. Ela é semlhante ao java, mas é integrada ao ecossistema Windows.

Exemplo:


         using system;
         
         class program {
            static void main() {
                console.writeline("Oi, mundo");
              }
          }
