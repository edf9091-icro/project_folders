Front-end na prática

Neste folder você vai entender como HTML, CSS e JavaScript trabalham juntos para criar algo simples, mas real: um botão funcional. Cada parte tem uma responsabilidade diferente.

HTML Estrutura

O HTML é responsável por criar os elementos que aparecem na tela. Aqui ele cria o botão e o espaço onde o texto será exibido. 


<button id="btnMensagem">Clique aqui</button>
<p id="mensagem"></p>

CSS Visual

O CSS define como o botão e o texto vão aparecer. Cores, tamanhos, bordas e efeitos visuais são controlados aqui. 


button {
  background: purple;
  color: white;
  border: none;
  padding: 10px 16px;
  border-radius: 6px;
  cursor: pointer;
}

#mensagem {
  margin-top: 10px;
  color: cyan;
}

JavaScript Ação

O JavaScript conecta o comportamento ao botão. Ele detecta o clique e altera o conteúdo da página. 


const botao = document.getElementById("btnMensagem");
const texto = document.getElementById("mensagem");

botao.addEventListener("click", () => {
  texto.textContent = "Você acabou de usar JavaScript!";
});
