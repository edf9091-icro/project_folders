Programação: base para iniciantes

Este projeto foi realizado para organizar fundamentos basicos da programação de forma simples, progressiva e acessivel. O foco principal é auxiliar quem esta iniciando e deseja construir uma base solida para melhor desenvolvimento pessoal.

Cada folder a seguir apresenta de maneira direta um conceito essencial, com exemplos que facilitem o desenvolvimento alheio.

Para quem é este material??

Separei este conteudo voltado para:

  • iniciantes em programação
  • interessados na area de tecnologia e programação

Como organizei o conteudo

A organização dos folders foi pensada de maneira ordenada, que facilite a compreensão: construção de base, estrutura de código, leitura visual e navegação.

abaixo estão os folders disponiveis:

  • 1 - Introdução à programção
  • 2 - linguagens de programção
  • 3 - programação web
  • 4 - base de mercado
